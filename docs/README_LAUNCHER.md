## [INFO] Troubleshooting

- **UnicodeEncodeError**
  > Imposta la console su UTF-8:
  >
  > ```sh
  > chcp 65001
  > ```
- **streamlit: command not found**
  > Usa:
  >
  > ```sh
  > python -m streamlit run ...
  > ```
